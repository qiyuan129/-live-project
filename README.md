## 构建和运行
1.将项目fork到本地，新建名为database的数据库后，使用给出的sql文件为此数据库导入相关表。
2.在eclipse或者IDEA中打开项目，导入数据库连接所需jar包，在程序中修改DBUtil数据库连接密码后即可运行。
（注意：推荐使用项目附带的两个jar包，否则可能存在时区问题导致功能出错）
